# Film Collection

## 📌 Project Description

This is an Angular application for managing a film collection.  
It allows users to browse, add, and manage movies.

The project was created as part of the RS School Angular course.

---

## 🚀 How to Run the Project Locally

### 1. Install dependencies

Open the project folder and run:

```bash
npm install
2. Start the development server
ng serve
3. Open the application

Go to your browser and open:

http://localhost:4200/

The application will automatically reload when you change source files.

🛠️ Build the Project

To create a production build, run:

ng build

The build artifacts will be stored in the dist/ directory.
